```bash
npx drizzle-kit push
```

## 2. Configure Remaining Infrastructure

- [ ] **Upstash Redis** — Create a database at [upstash.com](https://upstash.com) for rate limiting. Set `KV_REST_API_URL` and `KV_REST_API_TOKEN`.
- [ ] **Vercel Blob** — Create a store at [vercel.com/docs/storage/vercel-blob](https://vercel.com/docs/storage/vercel-blob) for file uploads. Set `BLOB_READ_WRITE_TOKEN`.
- [ ] **Sentry** (optional) — Set up at [sentry.io](https://sentry.io) for error monitoring. Set `SENTRY_DSN` and `NEXT_PUBLIC_SENTRY_DSN`.
- [ ] **Azure AI Foundry** — Set `AZURE_RESOURCE_NAME`, `AZURE_API_KEY`, `AZURE_API_VERSION`, and `AZURE_DEPLOYMENT_NAME` for AI features.

## 3. Environment Variables

Copy `.env.example` to `.env.local` and fill in all values. Then set the same variables in your Vercel project under **Settings > Environment Variables**.

## 4. Start Development

```bash
npm install
npm run dev
```
